# Dravida Restaurant Website

A luxury restaurant website featuring elegant split-scroll layouts, beautiful typography, and responsive design.

## Features

- **Parallax Hero Section** with GSAP animations
- **Split-scroll Content Sections** with sticky images
- **Private Dining Section** with parallax background
- **Newsletter Signup** with elegant form
- **Image Carousel** with manual scrolling
- **Responsive Design** optimized for mobile and desktop
- **Dark Theme** with gold accents
- **Custom Fonts**: Playfair Display SC & Lora

## Tech Stack

- React 19
- TypeScript
- Vite
- GSAP (animations)
- Tailwind CSS
- Lucide React (icons)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown (typically http://localhost:3001)

## Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

## Preview Production Build

```bash
npm run preview
```
